# Stella Synastry

Use this skill for two-chart comparison and relationship astrology.

## Workflow

1. Gather complete birth data for both people.
2. Run `scripts/stella_kerykeion.py synastry`.
3. Ground the reading in returned chart facts and mark unsupported validation types as `not checked`.

## Doctrine

Use modern synastry only and avoid deterministic relationship claims.
