# Stella Validation

Use this skill when the user asks whether astrology claims are supported or when a final answer needs fact checks.

## Workflow

1. Identify structured claims that depend on chart data.
2. Compare them against the Stella payload.
3. Mark each claim as supported, contradicted, `not checked`, or `needs review`.

## Scope

Validate direct payload claims and mark unsupported claims as `not checked` or `needs review`.
