# Stella Natal

Use this skill for natal chart requests and birth-chart interpretation.

## Workflow

1. Gather birth date, birth time, latitude, longitude, and IANA timezone.
2. Run `scripts/stella_kerykeion.py natal`.
3. Present chart facts first, then interpretation, then validation status.

## Doctrine

Use modern tropical natal interpretation with the configured house system. Keep chart facts, interpretation, and validation status distinct.
