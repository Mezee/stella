Stella Privacy Policy

Last updated: 2026-05-04

Stella is published by Creatorship.

What Stella Does

Stella processes astrology inputs such as birth date, birth time, location, timezone, chart requests, and user prompts in order to calculate chart data and generate astrology-related outputs.

Data Stella May Process

  • birth information entered by the user
  • chart calculation inputs and outputs
  • user prompts and generated responses
  • basic operational error data needed to diagnose failures

How Data Is Used

  • run astrology calculations
  • generate requested outputs
  • validate chart-based claims
  • maintain and improve plugin reliability

Sensitive Inputs

Birth data can be personally sensitive. Users should only submit data they are authorized to use.

Data Sharing

Creatorship does not sell user data through Stella.

If Stella relies on third-party runtime dependencies or hosting services, data may be processed by those services only as needed to operate the product.

Data Retention

Retention depends on the final deployment and hosting setup. Creatorship should publish the production retention policy before public release.

Security

Creatorship will take reasonable measures to protect data handled by Stella, but no system can guarantee absolute security.

Children's Privacy

Stella is not intended for children under 13.

Contact

For privacy questions, contact stella@creatorship.pro.
